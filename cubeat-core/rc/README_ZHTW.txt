系統需求
=========================================================
作業系統：Windows XP / Vista / 7、Mac OSX 10.6 以上、Linux
  處理器：Intel Pentium 4 2.0GHz 以上
  記憶體：1GB 以上
  解析度：1280×720 以上
  顯示卡：支援 OpenGL 2 以上的顯示卡

  
基本操作
=========================================================
1P 玩家（紅色游標，預設使用滑鼠操作）
滑鼠左鍵：決定／射擊
滑鼠右鍵：取消／加速
滑鼠中鍵：暫停

2P 玩家
（藍色游標，使用第二支滑鼠、筆電觸控版，或鍵盤方向鍵操作）
滑鼠操作時配置與1P玩家相同，鍵盤操作時：
"," 鍵：決定／射擊
"." 鍵：取消／加速
"/" 鍵：暫停


遊戲模式與規則
=========================================================
玩家vs玩家、玩家vs電腦、電腦vs電腦、解謎模式 

1. 射擊可打掉不要的方塊或還原垃圾（白色）方塊。
2. 同色三顆以上直橫排列消除。
3. 排列消除會使上方方塊具有連鎖屬性（閃爍）；再使之排列消除即增加連鎖段數。
4. 在仍有方塊保有連鎖屬性的情況下，死亡倒數會停住。
5. 1p 盤面一定在左邊、2p 盤面一定在右邊。可以打對手盤面作為干擾。
6. 射擊與加速到一定程度都會造成過熱（無法動作）。


注意事項及已知問題
=========================================================
1. 本遊戲目前僅能支援 1280x720 以上的畫面解析度。

2. 雙滑鼠：接兩支滑鼠就可以公平地與朋友用同一台電腦對戰。建議遊戲開始前
   就接上雙滑鼠。如果遊戲開始後才接也可以，只是要切出畫面再切回來這樣。

3. 遊戲會鎖系統滑鼠，這是為了雙滑鼠功能做的設計，要切出遊戲請 Esc 或 Alt-Tab

Windows 版：

1. 啟動 CuBeat.exe 就能直接玩；支援雙滑鼠進行兩人對戰。

MacOSX 版：

1. 啟動 CuBeat.app 就能直接玩；若有遇上問題，請先更新到 OSX 10.6 版本。

2. 暫不支援雙滑鼠，目前正在設法解決中。

Linux 版：

1. 請確定有正常運作、更新到最新的 OpenGL driver。

2. 請先 apt-get libopenal1 (Ubuntu)，或確認有一正常運作之 OpenAL driver。

3. 於遊戲主目錄 ./CuBeat.sh 就能直接玩；若要設立桌面捷徑或啟動雙滑鼠，再
   執行 ./setup.sh。
   
4. 有使用桌面特效 (compiz / GNOME 的動態或效果) 者建議關閉特效，可能導致
   視窗外框消失或其他異常，目前似乎查無解決方案。
