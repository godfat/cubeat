CuBeat 測試版使用說明

官方網站：http://cubeat.game.tw
問題回報：http://cubeat.game.tw/devblog/?page_id=2 或 cubeat.tw@gmail.com

=========================================================
基本操作：

1P 玩家
（紅色游標，預設使用滑鼠操作）
滑鼠左鍵：決定／射擊
滑鼠右鍵：取消／加速
滑鼠中鍵：暫停

2P 玩家
（藍色游標，使用第二支滑鼠、筆電觸控版，或鍵盤操作）
滑鼠操作時配置與1P玩家相同，鍵盤操作時：
","鍵：決定／射擊
"."鍵：取消／加速
"/"鍵：暫停

（現在遊戲內還以 A鍵、B鍵 來提示，A = yes, B = no）

=========================================================
遊戲模式：玩家vs玩家、玩家vs電腦、電腦vs電腦、解謎模式

遊戲規則：
1. 同色三顆以上直橫排列消除、射擊可打掉不要的方塊或還原垃圾方塊。
2. 排列消除會使上方方塊具有連鎖屬性（閃爍）；再使之排列消除即增加連鎖段數。
3. 在仍有方塊保有連鎖屬性的情況下，死亡倒數會停住。
4. 1p 盤面一定在左邊、2p 盤面一定在右邊。可以打對手盤面作為干擾。

圖說請參考：http://cubeat.game.tw/rule.htm

=========================================================
注意事項

雙滑鼠：建議遊戲開始前就接上雙滑鼠。如果遊戲開始後才接也可以，
        只是要切出畫面再切回來這樣。

遊戲會鎖系統滑鼠，這是為了雙滑鼠功能做的設計，要切出遊戲請 Esc 或 Alt-Tab

Windows 版：
解開 zip 進資料夾點 CuBeat.exe 就能直接玩，雙滑鼠支援內建。

MacOSX 版：
直接執行 CuBeat.app 開始遊戲。若有遇上問題，請先更新到 OSX 10.6 版本，
暫不支援雙滑鼠。

Linux 版：
1. 請確定有正常的 OpenGL driver，而非軟體模擬的
2. 請先 apt-get libopenal1
解開後進去目錄，./CuBeat.sh 就能直接玩，若要桌面捷徑或啟動雙滑鼠再執行
./setup.sh。有使用桌面特效 (compiz / GNOME 的 animation) 者建議關閉特效。
